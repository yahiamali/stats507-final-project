# March Madness Prediction Project

This project predicts NCAA March Madness outcomes using machine learning.

## How to run
1. Open Yahia Ali - Final Project.ipynb
2. Run all cells from top to bottom

## Output
- full_tournament_predictions_2026.csv

## Notes
- Data sourced from Hugging Face dataset
- Model uses player-level data aggregated into team-level and matchup-level features
- Evaluation uses season-based split and cross-validation
